Inked & Styled Projects Application Front End

This project was bootstrapped with Create React App, using the Redux and Redux Toolkit TS template. This is the front-end for the web application Inked & Styled Projects.

The purpose of the application is to show beauty salons with their reviews, if users have a salon, they can register, edit or delete it, receiving reviews from other users who consume from their salons.
